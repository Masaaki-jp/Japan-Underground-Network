# Japan-Underground-Network (JSP)

## 概要 / Overview
本プロジェクトは、行政の「公営地下鉄リスト」には載らない、日本全国に眠る**「影の地下ネットワーク（鉄道トンネル、地下通路、貨物線等）」**を可視化するサバイバルDXプロトコルです。

低スペックなChromebook（RAM 4GB）環境でも、Google Colabのクラウド処理能力を活用することで、日本全土の膨大なインフラデータをハックし、生存圏を特定することが可能です。

## 発見された「生存圏」の事例
今回のプロトコル実行により、川崎市中原区下新城周辺において、一般的な路線図には描かれない**「武蔵野貨物線」の巨大な地下トンネル**を可視化することに成功しました。これは有事における極めて堅牢な防護空間となり得ます。

## 技術スタック / Tech Stack
- **Device:** ASUS Chromebook CM30 (RAM 4GB)
- **Runtime:** Google Colab (Python 3.12)
- **Data Source:** OpenStreetMap (Overpass API), 国土数値情報
- **Key Library:** GeoPandas, requests, json
